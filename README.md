Scope:
- Créer un script qui va tourner toutes les X minutes (cron) pour récupérer les contacts depuis Odoo et qui va insérer / mettre à jour / supprimer le(s) contact(s) dans une table dans une base de donnée de ton choix 
OU Faire des requêtes directement sur l'API d'Odoo sans passer par une database intermédiaire
- Créer une interface API (de ton choix: fastapi, flask, django,... ou encore mieux tu peux utiliser du .NET) qui permet de récupérer les contacts depuis la base de donnée directement (get contacts et get one contact by id).

Bonus:
- Sécuriser les accès à l'API
- Déployer l'app sur une solution d'hébergement
- CI lors du déploiement

Les infos nécessaires :
- Tu peux aussi facilement te créer un compte gratuit sur Odoo ou héberger une instance en local via docker.
